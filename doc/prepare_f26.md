Fedora 26
=========

### Development Prerequisities

``` bash
$ sudo dnf install -y \
    gcc \
    sqlite \
    libyaml \
    libyaml-devel \
    git \
    pandoc \
    python3-virtualenv \
    python3-tox

$ sudo pip install --upgrade autopep8
```

### Cloning

``` bash
$ git clone https://github.com/zonkylla/zonkylla.git
$ cd zonkylla
```
