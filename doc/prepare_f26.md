Fedora 26
=========

### Development Prerequisities

``` bash
$ sudo dnf install -y \
    gcc \
    sqlite \
    libyaml \
    libyaml-devel \
    git \
    pandoc \
    python3-virtualenv \
    python3-tox \
    python3-autopep8
```

### Cloning

``` bash
$ git clone https://github.com/celestian/zonkylla.git
$ cd zonkylla
```
